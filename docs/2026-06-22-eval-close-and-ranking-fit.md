# 경진대회 평가 마감(순위 고정) + 결과평가 등수표 가로스크롤 제거

날짜: 2026-06-22

## 1) 평가 입력 마감 → 순위 고정
학생 발표가 10위까지로 마무리되어, 더 이상 순위가 변동되지 않도록 **평가 입력을 마감**한다.
순위는 라이브 평가 데이터로 계산되므로, 입력을 닫으면 데이터가 고정되고 순위도 고정된다.

- `src/config/competition.ts` (신규)
  - `EVAL_CLOSED = true` — 마감 스위치(단일 진실 공급원). false로 바꾸면 다시 열림.
  - `EVAL_CLOSED_NOTICE` — 마감 안내 문구.
- `src/pages/CompetitionPreEval.tsx`, `CompetitionResultEval.tsx`
  - 상단에 🔒 **평가 마감 배너** 표시.
  - 점수 입력·종합평 입력 `disabled` 처리(`disabled || EVAL_CLOSED`).
  - 저장/삭제 버튼 자리에 "평가가 마감되었습니다(+내 평가 점수)" 안내로 대체.
- 집계표·등수표는 마감과 무관하게 **항상 조회 가능**(읽기 전용 열람).

> 메모: 결과평가 대상(사전평가 상위 10팀)도 사전평가 입력이 닫혀 데이터가 고정되므로 그대로 고정됨.

## 2) 결과평가 등수표 가로스크롤 제거
결과평가 등수표는 항목이 10개라 표가 넓어 가로스크롤이 생겼다. 컬럼 간격·최소폭을 좁혀
일반 화면 폭(컨테이너 ≈1180px, 관리자 콘텐츠 영역 포함) 안에 들어가게 조정.

- `src/components/RankingTable.tsx`
  - 항목(숫자) 컬럼 전용 `thNum`/`tdNum`: 패딩 축소(3px), 폰트 11~12.5px, **헤더 라벨 줄바꿈 허용**(긴 라벨도 좁은 폭).
  - 순위·팀·총점·평가 컬럼 패딩 축소, 총점 헤더 2줄(`총점/평균`).
  - 표 `minWidth` 산식을 빠듯하게 축소: `130 + (팀원 104) + 항목수×40 + 140`.
    - 결과평가(10항목): 공개 ≈730px / 관리자 ≈834px → 가로스크롤 없음.
    - 사전평가(5항목)도 동일 컴포넌트라 더 여유 있게 들어감.
  - 주제·팀원 컬럼은 줄바꿈(전체 표시) 유지.
