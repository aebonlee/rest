# 종강 마감 — 사이트 현 상태 동결

날짜: 2026-06-22 (종강)

## 요약
1기 과정 종강에 맞춰 **현재 상태로 사이트를 마감(동결)** 한다. 경진대회 평가는 마감되어
순위가 고정되었고, 결과는 집계표·등수표에서 계속 열람 가능하다. 추가 코드 변경 없이
현 배포 상태를 최종본으로 유지한다.

## 최종 상태
- **평가 동결**: `EVAL_CLOSED = true` (`src/config/competition.ts`)
  - 사전·결과평가 입력 읽기전용, 점수 제출·수정 불가 → 데이터·순위 고정.
  - 발표는 사전평가 상위 10팀까지로 종료.
- **결과 열람**: 집계표(`/competition/eval-summary`, `/result-summary`),
  등수표(`/competition/eval-ranking`, `/result-ranking`) 및 관리자 화면에서 상시 조회 가능.
- **배포**: main 최신 커밋 `5f52118` 기준 GitHub Pages 배포 성공(success). rest.dreamitbiz.com 라이브.
- **저장소**: 작업트리 clean, origin/main 동기화 완료.

## 이번 세션(2026-06-22) 주요 작업
- 사전/결과평가 **등수표 페이지** 신설(`/competition/eval-ranking`, `/result-ranking`).
- **관리자 사이드바** 등수표 메뉴 + 관리자 레이아웃 임베드(주제 줄바꿈·팀원 표기).
- 결과평가 대상 차순위 보정은 검토 후 **되돌림**(순위 변동 없이 상위 10팀 유지).
- **평가 입력 마감** 스위치(`EVAL_CLOSED`)로 순위 고정.
- 결과평가 등수표 **가로스크롤 제거**(컬럼 간격·최소폭 조정).

## 재개 시(다음 기수 등)
- 평가를 다시 열려면 `EVAL_CLOSED`를 `false`로 변경 후 푸시.
- 순위/대상 로직은 라이브 평가 데이터 기반(`projectEval`/`projectResultEval`)으로 동작.
