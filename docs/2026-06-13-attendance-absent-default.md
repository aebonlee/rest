# 2026-06-13 — 출석 매트릭스: 지난 수업일 미기록을 결석(결)으로 표기

## 변경
"6월 전체 출석 현황" 매트릭스에서 출(present)/지(late)/사(excused) 기록이 없는 칸을 그동안 점(`·`)으로만
표시했다. 요구사항: **출/지/사가 아니면 결석** → 지난 수업일의 미기록은 결석(`결`)으로 표기.

- `effStatus(pkey, d)` 헬퍼 도입(`AdminAttendance.tsx`):
  - 출/지/사/결 기록이 있으면 그 상태,
  - 없고 **지난 수업일**(`날짜 < 오늘`, 로컬/KST 기준)이면 `absent`(결),
  - **오늘·미래 수업일**은 미정(`''`)으로 두어 점(`·`) 유지 — 미래를 결석 처리하지 않음.
- 매트릭스 셀과 누계(`tally`) 모두 `effStatus` 기준으로 통일 → 결석 누계에 "미기록 결석"도 반영.

## 검증
- `tsc --noEmit` 통과, `npm run build` 성공.
