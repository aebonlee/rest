## 반영 범위 / 주의
- **반영됨**: TEAM_PROJECTS를 id로 읽는 화면 — 사전평가/결과평가 집계표(admin), 경진대회 사전·결과평가 페이지, 프로젝트 카드(getTeamProject).
- **별도 작업 필요(미반영)**:
  - 앱 갤러리/투표보드 카드의 **제목·팀원**은 라이브 DB `rest_*`의 `project_topic`/멤버를 직접 표시 → 코드가 아닌 DB 수정 필요.
  - 7·11팀 제목 변경으로 갤러리에서 `getTeamNoByTitle(project_topic)` 미스매치 시 아이콘/색이 기본값(🚀/파랑)으로 표시될 수 있음(기능 영향 없음, 표시만).
  - 투표 보드 번호 매칭(boardOrder.ts)·projectTopics.ts·projectDetails.ts는 과거 투표 문자열과 연동되어 의도적으로 미변경.
- DB 측까지 동기화가 필요하면 SQL Editor에서 `project_topic` 업데이트로 후속 처리 가능.
